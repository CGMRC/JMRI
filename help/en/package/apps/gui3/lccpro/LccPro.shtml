JMRI: LccPro

Since JMRI 5.11.1 Note that LccTools is a work in progress. Feedback welcome! Help us improve this by telling us what would make it more useful to you.

LccTools is a JMRI application that focuses on configuring your LCC network.

The main LccTools window is a table showing all the LCC nodes on your network. Each row shows information about a single LCC node. If the node supports it, there are "Configure" and "Upgrade" buttons that will open functional windows for configuring the node and upgrading its firmware respectively.

When you select a row, more information about that node is shown on the lower left.

Across the top is a tear-off set of buttons that will open other LCC tools, including

  • The Event Table
  • The LCC Traffic Monitor
  • The STL Editor
It also includes a "Refresh" button that will reload the table and a "Help" button that will bring you to this page.

At the bottom of the window is a status bar containing the name of the currently configured connection, and an indicator of whether that LCC connection is currently active with traffic or not.

Because LccTools is a work in progress, the lower right region is not populated yet. Suggestions about what to put there would be appreciated!
