OpenLCB Hub Control


The "OpenLCB Hub Control" tool enables a physical OpenLCB connection to be shared on a Network Connection.

The window displays the IP address and port of the Hub, along with client connection and disconnections.

If the IP addresses is flagged as a Loopback or LinkLocal, this is indicated.

The OpenLCB Hub can be opened by adding as a system startup action.

This page can be opened from the Hub Control via Help > Window Help .

Line Termination

You should not normally need to change this, however if you do so, you are advised to restart JMRI.

Line Termination Enabled

Default for OpenLCB Hub Control.

Hub receives Frames whenever a line termination is received in the network stream.

Hub sends CAN Frames in the GridConnect Serial format, followed by a line termination character ( CR/LF ).

Line Termination Disabled

Hub reads the GridConnect content between the start of the Serial Format ":", and end of format ";" characters.

Hub sends CAN Frames in the GridConnect Serial format, with NO line termination character.

The Grid Connect Serial specification is the documentation for their CANUSB Interface, see section 2.7.1 Message String Syntax in the pdf at gridconnect.com.
Line terminations can be enabled / disabled in this USB devices output via a device setting.
